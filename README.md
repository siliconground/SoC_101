# SoC 101
Xilinx Zybo Z-20 Reference Board를 활용하여 SoC의 기본을 학습합니다

## 🫡 어서오세요
환영합니다!
SoC 101에서는 Digilent사의 Zybo Z-20 Reference Board를 활용하여 이제껏 쌓은 HDL기반의 회로 설계 기술을 활용하여 기초적인 SoC를 구현해 보아요!
지금껏 배운 지식을 마치 레고 블럭처럼 활용하여 하나씩 구성요소를 만들다 보면, 어느새 SoC라는 거대한 집이 완성된 것을 볼 수 있을 거에요.

## 🍕 구성은 다음과 같아요
  
1. **HDL Modeling(Fast Track)**
* SoC라는 집을 짓기 위한 기본적인 회로 블럭들을 빠르게 점검해요. 조합회로는 물론이거니와 순차회로, 그리고 Finite State Machine(FSM)을 가지고 놀다 보면 어느새 SoC 구현을 위한 기본기가 단단해 지는 것을 느낄 수가 있어요.

2. **SoC Component**
* SoC를 구현하는 최소한의 주요 구성요소를 알아보고 구현해 보아요. 주요 요소는 다음과 같아요: 
  *  8-bit MCU의 구현과 Porting
  *  각종 통신 Interface의 구현(Peripherals)
  *  Memory 와 FPGA상의 BRAM Control

## 🔐 실무 개발환경을 고려하여 구성했어요
* ASIC이든 FPGA는 본인이 원하는 방향으로 실습을 할 수 있도록 개발 환경을 구성했어요. 상용 솔루션을 사용하여도 좋고, 혼자서 연습할때는 무료 소프트웨어를 사용해서 충분하게 즐겁게 학습할 수 있어요.
